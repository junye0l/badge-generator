# 소개

<p>깃허브 프로필을 생성하면서 느꼈던 번거로움들을 조금이나마 해결하고자 웹사이트를 만들어봤습니다.</p>

<p>추후에 다른 위젯들도 편리할게 추가할 수 있도록 확장해 나갈 예정입니다.</p>

# 설명

<p>웹사이트에 접속하여 맘에드는 기술 뱃지를 확인하여 마크다운 코드를 복사하여 본인 깃허브 리드미에 쉽게 적용할 수 있도록 제작했습니다.</p>

<p>미리보기에 나와있는 기술스택을 제외한 나머지는 출처에 해당하는 깃허브 레포지토리에 방문하여 확인해주시면 감사하겠습니다 🙇🏻</p>
